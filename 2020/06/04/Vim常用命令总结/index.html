Vim common commands

Posted on 2020-06-04 Edited on 2020-06-29

Getting Vim on your own computer

Opening files

Editing files

press i to enter insert mode, and insert mode

Saving files

enter command mode (press : after entering normal mode) and type w

Closing Vim

enter command mode and type q

Different modes

Normal mode

  • Every time you open Vim, you will start in normal mode.
  • press ESC to enter normal mode

Insert mode

  • press i to enter insert mode, but more useful ways:
  • i : start inserting text where the cursor is
  • a : start inserting text right after the cursor
  • I : start inserting text at the beginning of the line
  • A : start inserting text at the end of the line
  • o : insert a new line after the current line and enter insert mode

Command mode

  • press : to enter command mode

Visual mode

  • In this mode, you can highlight multiple letters or lines and manipulate them
  • v : enters regular visual mode
  • V : highlights entire lines at a time
  • navigation : such as hjkl, w
  • deleting and yanking : d will delete all highlighted text; y will yank all highlighted text. Both then return to normal mode.

Navigating

  • h : left
  • j : down
  • k : up
  • l : right
  • word:
    • w : moves forward to the begging of the next word
    • b : move backward to the begging of the previous word
  • lines:
    • 0 : moves to the begging of the line
    • $ : moves to the end of the line
  • document:
    • gg : moves to the top of the line
    • G : moves to the bottom of the file

Undo and redo

  • u : enter normal mode an press u to undo, you can type multiple times
  • Ctrl-r: enter normal mode and press Ctrl-r to redo, multiple times too

Deleting text

The x command deletes a single character:

  • x deletes the character right underneath the cursor
  • X deletes the character right before the cursor

Using d provides more flexibility and range:

  • dd deletes the entire line
  • dw deletes the rest of the current word, starting from the cursor
  • D deletes from the cursor to the end of the line

Copying text

  • yy : copy the entire line
  • yw : copy the rest of the current word, starting from the cursor

Pasting text

  • p : pastes copied text right after cursor
  • P : pasted copied text right before cursor

Searching

  • /... : ... is a pattern, so that you can type anything you want to search
    • n : moves forward through all matches
    • N : moves backwards through all matches
  • :42 : so you can jump to line 42

Configuration
